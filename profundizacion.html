Bienvenido

Este sitio fue diseñado utilizando Sass (SCSS) para aplicar variables, anidación e interpolación.

¿Qué se aplicó en este proyecto?

  • Variables: Se usaron 5 variables Sass para definir colores, tipografía y espaciados, mejorando la consistencia del diseño.
  • Anidación: Permite organizar el código de forma jerárquica, reflejando la estructura del HTML.
  • Interpolación: Se emplearon 2 casos para generar nombres de clases dinámicamente, facilitando la reutilización del código.
  • Mixin: Se creó un mixin para los botones, que permite aplicar estilos coherentes en diferentes partes del sitio.
  • Compilación automática: Sass genera el archivo style.css para que el navegador interprete correctamente los estilos.

Con el uso de Sass, el código CSS es más limpio, escalable y fácil de mantener, cumpliendo los criterios del trabajo autónomo.
